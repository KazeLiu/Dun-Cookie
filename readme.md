# 安装方法
## Chrome安装方法
> - Chrome应用商店
> - 下载zip通过开发者模式加载已解压的扩展程序
> 
## Firefox安装方法
> 1. 先解压到任意文件夹
> 2. 地址栏输入 about:debugging ，回车
> 3. 点击“此Firefox”->临时载入扩展
> 4. 随便在解压后的插件文件夹里面点开任意文件即可加载
